New Forest Escapes (in progress)

New forest escapes is a ongoing group web applicaiton that aims to allow the agency an easier time renting, and charging customers for an excess of their energy usage. It's a build in progress and it would be build using an MERN framework. The goal of the project is to create an incetive for guests to be mindful of their energy usage, and allow the homeowner to track this energy usage for every guest.
